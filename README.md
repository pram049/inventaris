# Inventaris

Aplikasi Inventaris Barang berbasis web menggunakan Laravel.

## Fitur
- Manajemen kategori
- Manajemen produk
- Manajemen supplier
- Transaksi barang
- Dashboard laporan

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap

## Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Copy `.env.example` menjadi `.env`
4. Jalankan `php artisan key:generate`
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
